#  Rabbitmq-javascript-nodejs

  

***A pub/sub program using RabbitMQ and Node.js***

  

**Library Dependencies,**

 1. Install RabbitMQ using https://www.rabbitmq.com/download.html
 2. Install Erlang https://www.erlang.org/downloads

## Commands

     - npm i
       
     
     - node producer.js
    
       
    
     - node consumer-1.js
    
       
    
     - node consumer-2.js
